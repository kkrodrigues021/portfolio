# 🚀 Portfólio Corporativo | Kaiky Rodrigues

Este é o Dossiê Executivo de Kaiky Rodrigues. Desenvolvido com foco em **Tech Brutalism**, este portfólio atua como uma plataforma de apresentação B2B para soluções em TI, engenharia de software e infraestrutura corporativa.

## 🌐 Demonstração

▶️ [Acesse o Dossiê](https://www.google.com/search?q=https://kkrodrigues021.github.io/portfolio)

## ✨ Características Técnicas

* **Design Brutalista:** Foco em hierarquia visual, tipografia e grids de alta performance.
* **Interface B2B:** Estrutura focada em conversão para consultoria e infraestrutura.
* **Sistemas Integrados:** Painel de controle visual com projetos reais de backend e front-end.
* **Animações de Alta Performance:** Scroll Reveal, Ticker infinito e botões magnéticos.
* **Histórico Corporativo:** Dossiê de carreira integrado (TIVIT, Atos, Grupo MV3).
* **SEO & Performance:** Otimizado para renderização rápida e legibilidade técnica.

## 🖥️ Executando Localmente

1. Clone o repositório:
```bash
git clone [https://github.com/kkrodrigues021/portfolio.git](https://www.google.com/search?q=https://github.com/kkrodrigues021/portfolio.git)
```
2. Acesse a pasta:
```bash
cd portfolio
```
3. Use a extensão **Live Server** (VS Code) para abrir o `index.html`.
